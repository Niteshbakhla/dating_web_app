# Dating Web App

## Description
A full-stack MERN dating application with authentication, profile creation, and matching features.

## Tech Stack
- MongoDB
- Express.js
- React
- TanStack Query
- Node.js

## How to Run Locally
1. Clone the repository
2. Run `npm install` in both client and server folders
3. Create a `.env` file with required environment variables
4. Run `npm run dev`
